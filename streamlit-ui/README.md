# Streamlit UI

- Command to Run - `streamlit run main.py --server.port=7860 --server.address=0.0.0.0`

## Main Features Used

- `st.write()`
- `st.session_state`
- `st.sidebar`
- `streamlit_option_menu from option_menu`
- `streamlit_javasript as st_js` - to get browser info
- `st.experiment_rerun()`
- `st.container`
- `st.expander`
- `st.button` with `on_click=`
